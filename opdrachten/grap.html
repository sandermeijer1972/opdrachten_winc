API fun time
Bij deze opdracht gebruik je een API naar keuze: eentje met grappen, eentje over Pokemon of eentje met uitspraken van Donald Trump. Ik heb gekozen voor de grappen. Daarna heb ik zelf ook nog wat API's opgezocht om mee te werken, zoals bijvoorbeeld eentje van de NASA.

icanhazdadjokes

Dit is dus de API met grappen. Elke keer krijg je een willekeurige grap. De vereisten voor de opdracht:
  • als gebruiker wil ik op de pagina uitgenodigd worden om op een knop te klikken
  • na het klikken zal er 1 grap te zien zijn

NASA

Bij de API van NASA krijg je dagelijks een foto vanuit de ruimte. Deze kan je met een klik op de button tevoorschijn halen, met een beschrijving van wat er op de foto te zien is.

andere API's

Om ook de voorgaande leerstof niet kwijt te raken, heb ik bij deze opdracht ook veel gebruik gemaakt van technieken die al weer een beetje naar de achtergrond geschoven waren. Op de pagina zelf wijst het zich vanzelf wat je te zien krijgt als je op een bepaalde knop klikt. Ik heb gebruik gemaakt van API's met honden, katten, panda's, de datum, beledigingen, wereldsteden, hilarische Trump-tweets en gerechten.

kleine error

Bij knop 4 zit een kleine error, waardoor het niet altijd werkt. Dit is op te lossen door het volgende te doen:
  • rechtermuistoets, dan kies inspecteren
  • er verschijnt een blok naast je scherm
  • bovenin dat blok zie je staan "Elements"
  • daarnaast staat "console", selecteer die
  • dan zie je iets met rode letters
  • daarbij staat ook iets van "cors-anywhere.heroku"
  • klik daar op, er opent een nieuw scherm
  • klik daar op "request temporary access to the demo server"
  • knop 4 moet het dan weer doen

Maar er is inmiddels ook een link bij knop 4 geplaatst om toegang te verkrijgen.
