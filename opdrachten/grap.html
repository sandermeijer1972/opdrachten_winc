API fun time
Bij deze opdracht gebruik je een API naar keuze: eentje met grappen, eentje over Pokemon of eentje met uitspraken van Donald Trump. Ik heb gekozen voor de grappen. Daarna heb ik zelf ook nog wat API's opgezocht om mee te werken, zoals bijvoorbeeld eentje van de NASA.

icanhazdadjokes

Dit is dus de API met grappen. Elke keer krijg je een willekeurige grap. De vereisten voor de opdracht:
  • als gebruiker wil ik op de pagina uitgenodigd worden om op een knop te klikken
  • na het klikken zal er 1 grap te zien zijn

NASA

Bij de API van NASA krijg je dagelijks een foto vanuit de ruimte. Deze kan je met een klik op de button tevoorschijn halen, met een beschrijving van wat er op de foto te zien is.

andere API's

Op de pagina zelf wijst het zich vanzelf wat je te zien krijgt als je op een bepaalde knop klikt. Ik heb gebruik gemaakt van API's met honden, katten, de datum en gerechten.
