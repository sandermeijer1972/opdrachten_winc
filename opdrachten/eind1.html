eindopdracht HTML / CSS

doel van de opdracht:

Jouw HTML/CSS kennis bundelen tot een website waar je trots op bent.

minimum vereisten:

  • gebruik een duidelijk onderwerp voor jouw website
  • gebruik meerdere pagina's in jouw website
  • gebruik semantische HTML elementen op de juiste plaatsen met de juiste betekenis
  • gebruik CSS in een losse stylesheet (dus geen inline CSS
  • laat zien dat je het box-model onder de knie hebt en gebruik op minstens 2 plaatsen padding, margin en borders
  • gebruik minstens 1 plaatje
  • gebruik minstens 2 links
  • gebruik MediaQueries, zodat je website er anders uitziet op mobiel en op desktop
  • gebruik zowel CSS-grid als flex-box om de elementen te positioneren op je website
  • gebruik minstens 1 formulier
  • je website is mobile-first

bonus:

  • gebruik een embedded YouTube video
  • gebruik mooie op elkaar afgestemde kleuren
  • maak extra mediaqueries voor tablet
  • gebruik externe libraries zoals Google Fonts
  • maak een uitgebreid formulier met veel verschillende soorten inputvelden
